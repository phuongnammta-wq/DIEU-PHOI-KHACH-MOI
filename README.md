# Tra cứu ghế khách mời 24.4

## Các file chính
- `index.html`: giao diện web
- `styles.css`: giao diện điện thoại
- `app.js`: logic tìm kiếm
- `guests.js`: dữ liệu khách mời
- `DANH_SACH_KHACH_MOI_TRA_CUU_GHE_24_4.csv`: dữ liệu nguồn CSV

## Cập nhật lên GitHub
1. Giải nén file zip.
2. Vào repo GitHub cũ của bạn.
3. Chọn **Add file** > **Upload files**.
4. Kéo toàn bộ file trong thư mục này vào để ghi đè bản cũ.
5. Bấm **Commit changes**.
6. Chờ GitHub Pages cập nhật.

## Link GitHub Pages
Nếu repo của bạn đang bật GitHub Pages, link web vẫn giữ nguyên sau khi ghi đè file.
